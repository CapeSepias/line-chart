Versatile charts for AngularJS

What's that ?

Built on top of D3.js and AngularJS, n3-charts.line-chart directive creates beautiful charts in AngularJS applications. It is simple and interactive.

How to use ?

The GitHub page contains an exhaustive, comprehensive guide to use the line-chart directive. For several use cases, please check the examples section.
