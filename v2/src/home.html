n3-charts

Awesome charts for Angular

Installation

We've chosen npm as our primary distribution system. The reason is simple: Bower forced us to version the build files, which was annoying when reviewing/merging pull requests. We don't like annoying things.

The library is still in beta, which means it's not accessible through npm without specifying its version. Install it using the following command:

$ npm install n3-charts@2.0.0-beta23 --save-dev
